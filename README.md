# About

This is a simple TODO list application developed by NextJS and Typescript.<br/>
Redux-toolkit is used for state management.

## Available Scripts

Start the DEV server
```bash
npm run dev
```
Build & Start the PROD server
```bash
npm run build
npm start
```
Run unit tests
```bash
npm test
```
